# Changelog - 2025-12-05 23:51 (Europe/Helsinki)

## Changed

### Post Tag Display Enhancement

- **Updated post template to display all tags instead of only primary tag**
  - Modified `post.hbs` to loop through all tags using `{{#foreach tags}}` helper
  - Previously only the primary tag was displayed; now all tags are shown side by side

- **Added slash separators between tags**
  - Implemented separator display between tags (e.g., "Guides & Resources / Finland / AnotherTag")
  - Added `gh-article-tag-separator` span element with "/" character
  - Separator appears between tags but not after the last tag using `{{#unless @last}}` helper

- **Enhanced tag styling with proper spacing**
  - Added `gh-article-tags` wrapper class with flexbox layout
  - Configured `align-items: center` for proper vertical alignment
  - Added `margin: 0 8px` to separator for consistent spacing on both sides
  - Set separator opacity to 0.5 for subtle visual appearance
  - Removed gap property from wrapper as spacing is now handled by separator margins
